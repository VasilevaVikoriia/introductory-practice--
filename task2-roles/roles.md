# Роли пользователей платформы (Битрикс24)

**Платформа**: Битрикс24 — корпоративная экосистема для управления бизнесом (CRM, задачи, чаты, документы, порталы, телефония, склады).

---

## Роль 1. Администратор портала

| Параметр | Описание |
|----------|----------|
| **Цель на платформе** | Обеспечить бесперебойную работу портала, безопасность и удобство для всех сотрудников |
| **Функции** | 1. Настройка прав доступа и ролей для всех отделов / 2. Управление структурой компании (отделы, должности) / 3. Создание и настройка бизнес-процессов (роботы, сценарии) / 4. Контроль лицензии, тарифа и оплаты / 5. Мониторинг логов действий пользователей / 6. Настройка интеграций (1С, сайт, телефония, почта) / 7. Бэкап и восстановление данных портала / 8. Техническая поддержка сотрудников (сброс паролей, проблемы с доступом) |
| **Результат** | Стабильный, безопасный и производительный корпоративный портал |

---

## Роль 2. Руководитель отдела продаж (Руководитель CRM)

| Параметр | Описание |
|----------|----------|
| **Цель на платформе** | Управлять воронкой продаж, анализировать эффективность команды и прогнозировать выручку |
| **Функции** | 1. Просмотр сводной воронки продаж по всем менеджерам / 2. Постановка планов и KPI для каждого сотрудника / 3. Анализ отчётов по конверсии и средней сделке / 4. Настройка воронок и стадий для разных направлений / 5. Контроль выполнения задач менеджерами через канбан-доски / 6. Проведение планёрок через видеозвонки HD / 7. Назначение ответственных за лиды и сделки / 8. Экспорт отчётов в Excel/CSV для презентаций руководству |
| **Результат** | Выполнение плана продаж, прозрачная статистика и мотивированная команда |

---

## Роль 3. Менеджер по продажам

| Параметр | Описание |
|----------|----------|
| **Цель на платформе** | Эффективно вести клиентов от первого контакта до оплаты и повторных продаж |
| **Функции** | 1. Добавление лидов вручную или через импорт / 2. Конвертация лидов в сделки / 3. Отправка коммерческих предложений из CRM-карточки / 4. Выставление и отслеживание счетов / 5. Заполнение карточек клиентов (телефоны, email, реквизиты) / 6. Назначение напоминаний и задач по клиентам / 7. Приём входящих звонков через интегрированную телефонию / 8. Формирование акта сверки с клиентом |
| **Результат** | Закрытые сделки, выполнение KPI, база лояльных клиентов |

---

## Роль 4. Специалист техподдержки (Helpdesk)

| Параметр | Описание |
|----------|----------|
| **Цель на платформе** | Быстро и качественно решать проблемы клиентов, повышая лояльность |
| **Функции** | 1. Приём обращений через виджет Битрикс24 на сайте / 2. Создание тикетов из чатов и email-писем / 3. Приоритизация обращений по срочности / 4. Привязка тикетов к карточкам клиентов в CRM / 5. Переписка с клиентом внутри задачи / 6. Эскалация сложных инцидентов на разработчиков / 7. Использование базы знаний (справочников) для быстрых ответов / 8. Закрытие тикета с указанием решения и времени затрат |
| **Результат** | Высокий CSAT (удовлетворённость клиентов), низкое время реакции (SLA) |

---

## Роль 5. HR-менеджер

| Параметр | Описание |
|----------|----------|
| **Цель на платформе** | Управлять персоналом: от найма до адаптации и оценки эффективности |
| **Функции** | 1. Размещение вакансий на корпоративном портале / 2. Онбординг новых сотрудников (выдача доступов, вводные задачи) / 3. Ведение базы сотрудников с личными данными / 4. Согласование отпусков и больничных через бизнес-процесс / 5. Проведение опросов 360 градусов / 6. Настройка и выдача годовых премий через приложение "Деньги" / 7. Формирование штатного расписания и оргструктуры / 8. Публикация внутренних новостей и анонсов на ленте новостей |
| **Результат** | Эффективный найм, низкая текучка, прозрачный учёт персонала |

---

## Роль 6. Бухгалтер / Финансист

| Параметр | Описание |
|----------|----------|
| **Цель на платформе** | Контролировать финансы компании, управлять счетами и отчётностью через интеграцию с 1С |
| **Функции** | 1. Просмотр всех выставленных и оплаченных счетов в CRM / 2. Сверка платежей с выписками банка / 3. Формирование тов сверки с контрагентами / 4. Экспорт данных по сделкам в 1С:Бухгалтерию / 5. Контроль дебиторской задолженности по клиентам / 6. Создание счетов на оплату и счетов-фактур / 7. Управление расходами компании (закупки, аренда, зарплаты) / 8. Формирование отчётов о движении денежных средств (ДДС) |
| **Результат** | Прозрачная финансовая отчётность, отсутствие просроченной дебиторской задолженности |

---

## Роль 7. Руководитель проекта (Project Manager)

| Параметр | Описание |
|----------|----------|
| **Цель на платформе** | Планировать и контролировать выполнение проектов, управлять командой и сроками |
| **Функции** | 1. Создание проектов и группировка задач по этапам / 2. Назначение ответственных и установка дедлайнов / 3. Просмотр диаграммы Ганта (ленты времени) / 4. Контроль загрузки сотрудников через тайм-менеджмент / 5. Проведение ежедневных стендапов через видеозвонки / 6. Получение уведомлений о срывах сроков / 7. Формирование отчётов о прогрессе для заказчика / 8. Экспорт завершённых проектов в портфолио компании |
| **Результат** | Проекты сданы в срок, довольные заказчики, прозрачная загрузка команды |

## Примечание

**Совмещение ролей (один человек может выполнять несколько):**
- **Руководитель продаж** + **Менеджер по продажам** (в малом бизнесе)
- **Администратор** + **Специалист поддержки** (в небольших компаниях)
- **Бухгалтер** + **Финансист** (часто совмещены)
- **Руководитель проекта** + **Менеджер по продажам** (в проектных компаниях)

**Гибкость Битрикс24:** все роли настраиваются через **права доступа**. Можно создать любую уникальную роль под бизнес-процессы компании.
